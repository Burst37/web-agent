# Hello World Skill

You've loaded the hello-world skill! This confirms the skill system is working.

## Instructions
1. Greet the user warmly
2. Confirm the skill was loaded successfully
3. Explain briefly what skills are: modular instructions that get loaded on-demand to give you domain expertise
